HTML笔记（持续更新）

以下都是个人学习中整理的笔记（供大家参考）

HTML骨架格式

  • html标签：作用所有html中标签的一个根节点。
  • head标签：用于存放title，meta，base，style，script，link，注意在head标签中必须要设置的标签是title
  • title标签：让页面拥有一个属于自己的标题。
  • boby标签：页面的主体部分，用于存放所有的html标签：p，h，a，b，u，i，s，em，del，ins，strong，img

HTML标签关系

  • 嵌套关系:
    <head> <title> </title> </head>
  • 并列关系：
    <head></head>
    <boby></boby>

HTML常用标签

  • 标题标签：<h1>标题</h1>,<h2>,<h3>,<h4>,<h5>,<h6>
  • 段落标签：<p> 文本内容 </p>
  • 水平线标签：<hr />是单标签
  • 换行标签：<br />是单标签
  • div span标签：没有语义，<div> 内容 </div>，<span> 内容 </span>
  • 文本格式化标签
属性 属性值
<b></b><strong></strong> 粗体（XHTML推荐使用strong）
<i></i>和<em></em> 斜体（XHTML推荐使用em）
<s></s>和<del></del> 加删除线（XHTML推荐使用del）
<u></u>和<ins></ins> 加下划线（XHTML不赞成使用u）
  • 链接标签：<a href=“跳转目标” target=“目标窗口的弹出方式（_self _blank）”> 文本或图像 </a>
    1.外部链接，需要添加，如<a href=“http://www.baidu.com”> 百度 </a>
    2.内部链接，直接链接内部页面名称即可，如<a href=”1-标签排版.html”> 标签排版 </a>
    3.如果当时没有确定的链接目标，可将属性值定义为href=“#”，表示该链接暂时未空链接
    4.文本，图像，表格，音频，视频等都可添加超链接
  • 锚点定位：通过创建锚点链接，用户能够快速定位到内容
    <a href=“#banji”>班级名</a>
    <h3 id=“banji”>计算机网络技术</h3>
  • base标签：可以设置整体链接的打开方式，放在head标签中
    <head> <base target=“_blank”/> </head>
  • 注释标签：<!-- 注释内容 -->
  • 图像标签：<img src=“图像URL” />
属性 属性值 描述
src URL 图像的路径
alt 文本 图像不能显示时的替换文本
title 文本 鼠标悬停时显示的内容
width 像素（XHTML不支持%页面百分比） 设置图像的宽度
height 像素（XHTML不支持%页面百分比） 设置图像的高度
border 数字 设置图像边框
  • 相对路径
    1.图像文件和HTML文件位于同一文件夹，只需输入图像名称即可，如<img src=“1.jpg” />
    2.图像文件位于HTML文件的下一级文件夹，输入文件夹名和文件名，之间用/隔开，如<img src=“图片/风景/1.jpg ” />
    3.图像文件位于HTML文件的上一级文件夹，在文件夹名之间加../，如果上两级../../，如<img src=“../风景/1.jpg”/>
  • 绝对路径：“D:\web\img\1.jpg”，或完整的网络地址，例如http://www.itcast.cn/images/logo.gif
